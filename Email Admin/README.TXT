PROJECT TITLE: Simple Email Admin
PURPOSE OF PROJECT: Develop a simple email admin for creating password and email based on person's name and department.
VERSION or DATE:10/16/2018
HOW TO START THIS PROJECT: Download both the Email and EmailApp java files to run the project
AUTHORS: D. Holt
USER INSTRUCTIONS:Download both the Email and EmailApp java files to run the project in a java friendly program. There are some get methods in the Email.java file that can allow an alternate email, change of password, and change of mailbox capacity that you can put in the main EmailApp.java file. The Company variable can be changed to fit whatever company you want to use for this program.
